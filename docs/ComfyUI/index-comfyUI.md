## 簡介

**ComfyUI** 是一個強大且靈活的圖形化界面，用於運行與構建 Stable Diffusion 圖像生成工作流程。透過其節點 (Node) 系統，使用者可以直觀地建立圖像生成管道，並自由調整各種參數以獲得理想的結果。

作為一個開源工具，ComfyUI 提供了完全透明的代碼基礎，使用者可以自由查看、修改或擴展其功能，以滿足個人或專業需求。
